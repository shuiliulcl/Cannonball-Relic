# 咒语 Combo 设计清稿

本稿根据最新结论整理，作为后续实现依据。

## 保留边界

- 旧库只保留原本已有的人间大炮核心三段：`一级准备`、`人间大炮`、`发射`。
- 旧库里只允许 `梆梆` 作为关键词，派生两个隐藏大招：`你就说梆梆不梆梆` 和 `梆梆两拳`。
- 其他旧库咒语不再做拆解。
- 新内容里，`我去不早说` 拆成 `我去` + `不早说`。
- `你已急哭` 不做拆解，作为独立咒语或独立强力技能候选。

## 完整咒语设计

### 原有人间大炮核心

关键词/流程：`一级准备` -> `人间大炮` -> `发射`

设计结论：
- 这是原本已有核心玩法，继续保留。
- 不额外拆成碎片隐藏 Combo。
- 后续只需要保证新 Combo 能和人间大炮槽、装填、落地冲击形成联动。

### 你就说梆梆不梆梆

关键词：`梆梆`

触发条件：
- 玩家已获得关键词咒语 `梆梆`。
- 玩家念出完整句 `你就说梆梆不梆梆`。

关键词单独效果：
- 对最近敌人打一次冲击拳。
- 命中返还少量声能。

隐藏大招效果：
- 对附近敌人连续打出多段冲击拳。
- 每命中一个敌人，返还声能和大炮槽。
- 最后一拳把敌人向外打飞，撞到其他敌人时造成二次伤害。

差异化：
- 偏范围近战、偏资源返还。
- 比 `梆梆两拳` 更适合被包围时使用。

### 梆梆两拳

关键词：`梆梆`

触发条件：
- 玩家已获得关键词咒语 `梆梆`。
- 玩家念出完整句 `梆梆两拳`。

隐藏大招效果：
- 第一拳定向打断，击退正前方敌人。
- 第二拳重击，优先锁定精英、远程或静音敌人。
- 如果第二拳击杀目标，立刻返还一次 `梆梆` 的普通咒语费用。

差异化：
- 偏单体爆发、强打断、精英处理。
- 比 `你就说梆梆不梆梆` 更短、更快、更像斩杀技能。

### 我要验牌

关键词：`验牌`

触发条件：
- 玩家已获得关键词咒语 `验牌`。
- 玩家念出完整句 `我要验牌`。

关键词单独效果：
- 标记当前最危险敌人。
- 被标记敌人短时间受到伤害提高。

隐藏大招效果：
- 全场翻牌，危险敌人暴露弱点。
- 精英、远程、静音、高血量敌人被强制破甲。
- 被揭穿的敌人身上爆出“作弊牌”，造成锁定爆破。

玩法价值：
- 用来反打混乱局。
- 解决玩家不知道先处理谁的问题。

### 我去不早说

关键词：`我去`、`不早说`

触发条件：
- 玩家已获得 `我去` 和 `不早说`。
- 玩家念出完整句 `我去不早说`。

关键词单独效果：
- `我去`：短促闪避，带极短无敌。
- `不早说`：返还最近一次普通咒语的部分声能。

隐藏大招效果：
- 回溯玩家 2 秒前的位置和部分 HP。
- 敌人已受到的伤害保留。
- 刷新最近一次非隐藏 Combo 咒语。
- 如果回溯前刚受伤，原地留下一个“早说残影”嘲讽并爆开。

玩法价值：
- 允许玩家反悔一次操作。
- 适合高压、误判、被围时救场。

### 不讲不讲

关键词：`不讲`

触发条件：
- 玩家已获得关键词咒语 `不讲`。
- 玩家念出完整句 `不讲不讲`。

关键词单独效果：
- 打断最近一个蓄力敌人。
- 小范围沉默远程或静音怪。

隐藏大招效果：
- 展开拒绝沟通领域。
- 区域内敌人停止施法。
- 已存在飞行弹幕变成废纸落地。
- 玩家低血量时领域扩大，持续更久。

玩法价值：
- 明确反制远程、静音、蓄力敌人。
- 低成本好理解，适合早中期。

### 你已急哭

设计结论：
- 不做关键词拆解。
- 作为独立强力/乐子技能进入库。

独立效果：
- 让附近敌人集体红温。
- 红温敌人攻击目标混乱，远程弹幕有概率打歪。
- 红温期间敌人受到反伤提高。
- 如果玩家刚刚受伤后使用，额外获得护盾。

玩法价值：
- 这是一个情绪型嘲讽技能，不需要碎片铺垫。
- 适合做中后期强力咒语。

### 收到，收到

关键词：`收到`

触发条件：
- 玩家已获得关键词咒语 `收到`。
- 玩家连续说出 `收到，收到`。

关键词单独效果：
- 复制上一个普通咒语的一小部分效果。
- 没有上一咒语时回复少量声能。

隐藏大招效果：
- 复读咒语链最后两个普通咒语。
- 给复读咒语附加一次小爆破。
- 如果最后两个咒语类型不同，额外触发协同确认并返还声能。

限制：
- 不能复制隐藏 Combo。
- 不能递归复制 `收到，收到` 自己。

玩法价值：
- 让玩家主动经营咒语顺序。
- 是构筑型 Combo 的核心连接件。

### 不知道，我的身材很曼妙

关键词：`不知道`、`身材`、`曼妙`

触发条件：
- 玩家已获得 `不知道`、`身材`、`曼妙` 三个关键词。
- 玩家念出完整句 `不知道，我的身材很曼妙`。

关键词单独效果：
- `不知道`：敌人短暂丢失锁定，玩家获得一点无敌。
- `身材`：短时间缩小玩家受击判定，但不增加输出。
- `曼妙`：优雅侧滑，擦过弹幕时回复少量声能。

隐藏大招效果：
- 玩家进入 `曼妙判定` 状态。
- 受击判定显著缩小。
- 大部分贴边弹幕不算命中。
- 擦弹转化为声能和刀光反击。
- 每次成功擦弹叠一层 `自信`。
- 结束时按 `自信` 层数爆出一圈光环，击退并伤害敌人。

玩法价值：
- 标准三关键词高阶 Combo。
- 单个关键词很弱，但完整句非常强。
- 适合高手秀操作，也能给普通玩家救命窗口。

### 与其内耗，不如外耗

关键词：`内耗`、`外耗`

触发条件：
- 玩家已获得 `内耗` 和 `外耗`。
- 玩家念出完整句 `与其内耗，不如外耗`。

关键词单独效果：
- `内耗`：消耗一点 HP 或护盾，换取声能和短暂伤害提升。
- `外耗`：诱导附近敌人互相碰撞，造成少量伤害。

隐藏大招效果：
- 将玩家身上的压力线转嫁给怪群。
- 敌人互相拉扯、撞击、反伤。
- 玩家获得短暂免伤。
- 玩家血量越低，外耗冲击越强。
- 如果玩家刚被围住，直接撕开包围圈。

玩法价值：
- 适合低血反打和被围时开路。
- 风格上和“内耗/外耗”的语义很贴。

### 爱你老己，明天见

关键词：`老己`、`明天见`

触发条件：
- 玩家已获得 `老己` 和 `明天见`。
- 玩家念出完整句 `爱你老己，明天见`。

关键词单独效果：
- `老己`：回复少量 HP，获得一点护盾。
- `明天见`：挂一个延迟回复，数秒后生效。

隐藏大招效果：
- 获得一次 `今晚不死` 保险。
- 持续期间若受到致命伤，保留 1 HP。
- 震开近身敌人。
- 1 秒后回复大量 HP。
- 保险触发后，下一次普通咒语免费。

玩法价值：
- 作为稀有保命大招很清晰。
- 不抢人间大炮的输出定位，专注续命。

## 第一批实现建议

优先级从机制明确度和实现收益排序：

1. `梆梆` + 两个派生大招：旧库最容易承接，能马上丰富已有乐子技能。
2. `我去不早说`：回溯和退款辨识度很高。
3. `不讲不讲`：沉默/打断价值明确，适合补控制。
4. `不知道，我的身材很曼妙`：最符合三碎片隐藏 Combo，建议作为重点高阶设计。
5. `收到，收到`：构筑潜力高，但需要小心递归和复制规则。
6. `我要验牌`：适合做危险目标反制。
7. `你已急哭`：独立强力技能，适合后期投放。
8. `与其内耗，不如外耗`：后期生存反打。
9. `爱你老己，明天见`：稀有保命大招。
