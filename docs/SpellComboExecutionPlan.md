# 咒语 Combo 实现与美术执行文档

本文件对应 `docs/SpellComboDesignClean.md`，用于落地运行时代码和美术表现。实现目标是：乐子咒语统一走“关键词碎片能单独施放，完整句在碎片齐备后触发隐藏 Combo”的规则；完整句效果必须明显强于碎片相加。只有 `邪修` 这类难拆、短词、语义已经完整的技能保留为独立乐子技。

## 技术落点

- 运行时配置：`src/survivor/VoiceSurvivorGame.ts` 的 `SPELL_CONFIG`。
- 语音识别：沿用 `matchSpells`，隐藏 Combo 通过 `hidden: true` 获得更高匹配优先级。
- 抽卡入库：关键词碎片通过 `createBuffPool()` 进入升级池；隐藏 Combo 不进抽卡池。
- 触发检查：`castHiddenCombo()` 检查 `fragments` 是否全部已解锁。
- 表现层：复用 `addSpellRing()`、`addSpellFan()`、`addBurst()`、`fireRadialProjectiles()`，并补充少量状态逻辑。

## 美术表现

### 通用表现语法

- 关键词碎片：小半径 `addSpellRing`，颜色单一，文字短。
- 隐藏 Combo：大半径环形 VFX + 多颗粒爆发 + 状态文案。
- 控制类：冷蓝、纸白、断线感。
- 近身拳类：金黄、鞋印/拳印感，用连续爆点表现。
- 回溯类：青绿残影、倒带环、原地残影爆点。
- 精神压制类：红粉、抖动、红温标记。
- 自救类：暖白、邮戳、延迟回复光环。

## 风险与限制

- `收到，收到` 不能复制隐藏 Combo，也不能复制自己，避免递归。
- `我去不早说` 只回溯玩家状态，不回滚敌人死亡和伤害，避免高成本状态快照。
- `不知道，我的身材很曼妙` 通过缩小受击判定和擦弹处理实现，不改变真实角色贴图尺寸过多。
- `爱你老己，明天见` 只保存一次免死保险，触发后清空。

## 已落地代码映射

- `SPELL_CONFIG`：已新增第一批关键词碎片、`你已急哭`、隐藏 Combo；下一步需要把 `技能五子棋`、`你已急哭`、`当个事儿办` 从单句技能改为碎片组合。
- `castHiddenCombo()`：统一检查碎片是否齐备，再触发完整梗大招。
- `recordPlayerSnapshot()` / `castTooLateCombo()`：支撑“我去不早说”的玩家状态回溯。
- `effectivePlayerRadius()` / `updateEnemyShots()`：支撑“身材”“曼妙”和完整 Combo 的缩判定、擦弹回能。
- `hurtPlayer()`：支撑“爱你老己，明天见”的免死保险和下一普通咒语免费。
- `createBuffPool()`：关键词碎片进入抽卡池，隐藏 Combo 不进入抽卡池。
