# 咒语列表

这个文件是《人间大炮一级准备》的咒语登记表。后续新增、改名、删减任何咒语时，都要同步更新本文件，确保设计、代码和测试口径一致。

## 维护规则

- 新增咒语必须登记：名称、分类、声能/资源消耗、语音别名、解锁/投放阶段、实际效果。
- 如果咒语会进入左侧倒计时，分类写为“持续战斗 Buff”。
- 如果咒语是一次性爆发或乐子技能，分类写为“强力/乐子技能”。
- 如果咒语影响位移、防御、拾取、锁敌，分类写为“通用操作/救场”。
- 如果咒语属于人间大炮三段式，分类写为“人间大炮核心”。
- `src/survivor/VoiceSurvivorGame.ts` 中的 `SPELL_CONFIG` 是运行时咒语配置源：名称、消耗、分类、阶段、效果和语音别名都从这里派生。
- 新增咒语时优先加入 `SPELL_CONFIG`；`SPELL_NAMES`、`SPELL_COSTS` 和语音识别列表会自动派生，随后同步检查本文件的设计说明。
